# Bordered Header

Creates a vertically-centered horizontal line to the right of any content in the element.

<Canvas>
  <Story name="Bordered header">
    {`
      <h2 class="a-bordered-header">Bordered header</h2>
    `}
  </Story>
</Canvas>

## Customization

The space between the text and the border can be specified, and the border styles:

```scss
$gutter: var(design-token.get('size', 'l3');
$border: var(design-token.get('size', 's7') solid var(design-token.get('color', 'grayscale', 'light-2');
```

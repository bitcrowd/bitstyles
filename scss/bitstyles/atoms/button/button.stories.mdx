## Buttons with icons and text

**Note**: if you want a button with _only_ an icon, go look at [`Icon buttons`](/docs/atoms-button-icon--button-element)

These are buttons containing a visible text label next to an icon. The order of text/icon is dictated by the order of the elements, you do not need to add a class to change it.

<Canvas>
  <Story name="Button with Icon">
    {`

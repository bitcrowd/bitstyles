## Button

All the buttons produce a larger-than-text hit target, and by default react to `:hover`, `:focus`, `:active`, and `:disabled`. Visited states are not shown even if the button is an anchor, as “visited” doesn’t make sense for an action.

## Buttons with icons and text

**Note**: if you want a button with _only_ an icon, go look at [`Icon buttons`](/?path=/docs/atoms-button-icon--button-element)

These are buttons containing a visible text label next to an icon. The order of text/icon is dictated by the order of the elements, you do not need to add a class to change it.

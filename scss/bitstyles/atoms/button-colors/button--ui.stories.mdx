# UI Buttons

Buttons that look more like native UI than the other color-branded buttons, intended to be used for actions that do not need to be highlighted as the main action, such as pagination.

<Canvas>
  <Story name="UI">
    {`
      <button type="button" class="a-button a-button--secondary u-h6">
        Edit account
      </button>
    `}
  </Story>
  <Story name="UI [disabled]">
    {`
      <button type="button" class="a-button a-button--secondary u-h6" disabled>
        Edit account
      </button>
    `}
  </Story>
</Canvas>

<Canvas>

When using the buttons for dropdown menus, add the `aria-expanded` attribute instead, and toggle it between true/false to indicate state.

<Canvas>
  <Story name="Dropdown">
    {`
      <button type="button" class="a-button a-button--secondary u-h6" aria-controls="element-id" aria-expanded="false">
        <span>Dropdown</span>
        <svg width="14" height="14" class="a-icon a-icon--m" aria-hidden="true" focusable="false">
          <use xlink:href="${icons}#icon-caret-down"></use>
        </svg>
      </button>
    `}
  </Story>
  <Story name="Dropdown [expanded]">
    {`

          <use xlink:href="${icons}#icon-caret-down"></use>
        </svg>

    `}

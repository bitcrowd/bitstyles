# Large Nav Button

A button for your main navigation that’s rendered full-width, ideal for the “off-screen-canvas” (or “hamburger”) menu on small screens.

<Canvas>
  <Story name="Nav-large">
    {`
      <a href="/" class="a-button a-button--nav-large">
        Dashboard
      </a>
    `}
  </Story>
</Canvas>


One of the links may well be a link to the current page, and should be given an aria-current attribute: `aria-current="page"`:

<Canvas>
  <Story name="Nav-large [aria-current]">
    {`
      <a href="/" class="a-button a-button--nav-large" aria-current="page">
        Dashboard
      </a>
    `}
  </Story>
</Canvas>

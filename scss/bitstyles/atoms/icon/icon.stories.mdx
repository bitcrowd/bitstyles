# Icons

## SVG icon system

Icons are best implemented using inline SVG (see the examples below). Add a machine-readable text version of the icon next to it for accessibility (and SEO). the text can be made invisible (e.g. because of your design requirements) â€” use [`.u-sr-only`](/docs/utilities-sr-only--sr-only) to hide an element containing descriptive text.

The base icon size inherits from the surrounding text i.e. `1em`. Other sizes available by default (`s`, `m`, `l`, and `xl`) are based on the global rem-based sizing map.

## Customization

The available sizes and their classnames can be customized using `$bitstyles-icon-sizes`:

```scss
$sizes: (
  's': size.get('s4'),
  'm': size.get('m'),
  'l': size.get('l2'),
  'xl': size.get('l2') + size.get('s4'),
);
```

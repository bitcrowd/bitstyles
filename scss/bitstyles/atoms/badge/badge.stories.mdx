# Badge

Display tags, filters, or categorization.

## Variants

Badges are available in a range of color variations, and two size variations (when using the default configuration — see [Customization](#customization) below for details on how to change that).

Color variants:

- grayscale (default)
- brand-1
- brand-2
- danger
- warning
- positive

Size variants:

- base (default)
- small

All badges are displayed as `small` badges on small screens.

## Customization

The available color variants can be specified with `$color-variants`, where you pass the name of the variant and a Sass map of two keys (one for color, one for background-color) for each variant:

```scss
@use '~bitstyles/scss/bitstyles/atoms/badge' with (
  $color-variants: (
    'badge-name': (
      'color': palette.get('brand-1', 'light-4'),
      'background-color': palette.get('brand-1', 'dark-4'),
    ),
  )
);
```

Similarly for the size variants — override the `$size-variants` variable, with each breakpoint (or use `setup.$no-media-query` for the base styles) at the root, containing a size-variant name and Sass map with the vertical & horizontal padding:

```scss
@use '~bitstyles/scss/bitstyles/atoms/badge' with (
  $size-variants: (
    '#{setup.$no-media-query}': (
      '': (
        'padding-vertical': size.get('s6'),
        'padding-horizontal': size.get('l1'),
      ),
    ),
    'l': (
      '': (
        'padding-vertical': size.get('s4'),
        'padding-horizontal': size.get('l3'),
      ),
    ),
  )
);
```

## Responsive

## Customization

Card appearance can be customized by overriding the following variables:

```scss
$bitstyles-card-sizes: (
  'base': size.get('m'),
  'l': size.get('xl'),
);
$bitstyles-card-border-radius: size.get('m');
$bitstyles-card-box-shadow: shadow.to-box-shadow(
  shadow.generate(
    $color: rgba(palette.get('brand-1', 'base'), 0.025),
    $offset: size.get('s'),
    $blur-radius: size.get('l'),
    $layers: 6,
  )
);
```

The breakpoints the cards are available at can be customized by overriding the `$bitstyles-card-breakpoints` variable:

```scss
$bitstyles-card-breakpoints: ('m', 'l');
```

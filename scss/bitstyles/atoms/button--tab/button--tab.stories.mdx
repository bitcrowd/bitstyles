# Tab Buttons

Buttons for selecting which of several tabpanels will be shown. There are several aria-attributes here that will need to be tied to the tabpanels, and you should use a background color behind the tabs — see the [tab examples](/docs/ui-navigation-tabs--tabs) for both.

## Using anchor elements for the tabs

Ideally, these would be `<button>` elements, and the behavior handled using JS, but if necessary server-side routing using anchor elements can be used instead.

## Icon-only tab buttons

Tab buttons can be used with just an icon as a visible label (use `.u-sr-only` to provide an invisible text label too). Tab buttons make use of CSS custom properties to render the icon in a color different to the button text — see [Button colors](/docs/atoms-button-colors--page) for details on how that works.

## Customization

Tab buttons are a combination of a color and a shape button variant, so you may need to adjust both aspects if you want to customize these to match your designs. See [Button shapes](/docs/atoms-button-shapes--page) and [Button colors](/docs/atoms-button-colors--page) for details.

# Dangerous Buttons

A button that performs any action the user should be wary off, most often something destructive to data.

<Canvas>
  <Story name="Danger">
    {`
      <a href="/" class="a-button a-button--danger">
        Destroy
      </a>
    `}
  </Story>
</Canvas>

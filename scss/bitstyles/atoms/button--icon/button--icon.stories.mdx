# Icon Button

A specialised button type where the visible content is intended to be only an icon. Commonly used for `+`, `<`, and `>` buttons, share buttons (facebook, twitter etc.).

Always provide:

- **a `title` attribute**. When a user hovers their pointer over the button, they’ll be told what it does (icons alone are not always clear).
- **a `sr-only` label** containing descriptive text next to the icon, so the button will have valid text to use as button label (see examples). This dictates whether people with visual impairments (whether temporary or permanent) can use our UIs.


Here’s a base icon button:

# Icon Button

## A button with _only_ an icon as its visible content

A specialised button type where the visible content is intended to be only an icon. Commonly used for `+`, `<`, and `>` buttons, share buttons (facebook, twitter etc.).

Always provide a `title` attribute, and a `visuallyhidden` span containing descriptive text next to the icon so the button has valid text to use as button label (see examples). This dictates whether people with visual impairments (whether temporary or permanent) can use our UIs.

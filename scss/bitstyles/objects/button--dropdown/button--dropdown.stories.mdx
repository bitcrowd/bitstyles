# Dropdown Button

A button that controls the visibilty of another component, normally a dropdown menu or similar. As well as some visual differences to a normal button (as they perform different actions) there are some aria attributes to ensure the button is connected to the element it controls.

When using this component:

- replace the `aria-controls` attribute’s value with the actual ID of the element it’s controlling
- update the `aria-expanded` attribute when the user clicks this button to show/hide that other element. Set it to `true` or `false` to reflect its current state.

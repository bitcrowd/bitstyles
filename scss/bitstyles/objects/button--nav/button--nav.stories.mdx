# Nav Button

A button object that looks like a link in your main navigation (and must be an acnhor element), but has the niceties of our button classes.

Here’s a base nav button. Note that it expects to be rendered on a dark background. Most likely this will be a navbar.

<Canvas>
  <Story name="Button--nav">
    {`
      <div class="u-bg--black-80 u-padding-m">
        <a href="/" class="o-button o-button--nav">
          Dashboard
        </a>
      </div>
    `}
  </Story>
</Canvas>

One of the links may well be a link to the current page, and should be given an aria-current attribute: `aria-current="page"`:

<Canvas>
  <Story name="Button--nav [aria-current]">
    {`
      <div class="u-bg--gray-90 u-padding-m">
        <a href="/" class="o-button o-button--nav" aria-current="page">
          Dashboard
        </a>
      </div>
    `}
  </Story>
</Canvas>

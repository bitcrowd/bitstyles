# Large Nav Button

A button for your main navigation that’s rendered full-width, ideal for the “off-screen-canvas” (or “hamburger”) menu on small screens.

<Canvas>
  <Story name="Button--nav-large">
    {`
      <div class="u-bg--black-80 u-padding-m">
        <a href="/" class="o-button o-button--nav-large">
          Dashboard
        </a>
      </div>
    `}
  </Story>
</Canvas>


One of the links may well be a link to the current page, and should be given an aria-current attribute: `aria-current="page"`:

<Canvas>
  <Story name="Button--nav-large [aria-current]">
    {`
      <div class="u-bg--gray-90 u-padding-m">
        <a href="/" class="o-button o-button--nav-large" aria-current="page">
          Dashboard
        </a>
      </div>
    `}
  </Story>
</Canvas>

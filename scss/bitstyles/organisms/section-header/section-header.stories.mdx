## With text only

In this section, the user interface content features a simple and clean design with text-only elements. The main focus is on the heading, which is presented prominently at the top. The heading text is accompanied by a possible label, denoted by a small span element that appears next to the heading. The absence of buttons or other interactive elements allows for a straightforward and uncluttered presentation, making it ideal for conveying essential information or introducing a specific topic.

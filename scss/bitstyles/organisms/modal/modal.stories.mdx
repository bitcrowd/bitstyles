# Modal

**Needs JS to function**

Set modal size using the usual layout classes e.g. `.a-content`. At smaller screensizes the modal becomes fullwidth.

Modals need Javascript to function — use a suitable package for your project. The examples here use [A11y-dialog.js](https://github.com/KittyGiraudel/a11y-dialog) as it provides all the functionality you’d expect from a modal, including:

- keyboard navigability
- focus trapping
- relevant aria attributes

## Informational

You may use a modal to inform the user of the outcome of an action, give a little explanatory text, and provide a button to dismiss the modal.

## Modal dialog

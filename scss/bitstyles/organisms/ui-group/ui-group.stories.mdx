# UI group

A collection of interactive elements that are closely-related can be visually bound together.

Commonly, this layout is used for buttons.

<Canvas>
  <Story name="UI group - buttons">
    {`
      <ul class="o-ui-group u-flex u-list-none">
        <li>
          <button type="button" class="a-button a-button--secondary o-ui-group__item u-border-radius-0-tr u-border-radius-0-br">First</button>
        </li>
        <li>
          <button type="button" class="a-button a-button--secondary o-ui-group__item u-border-radius-0">Middle</button>
        </li>
        <li>
          <button type="button" class="a-button a-button--secondary u-border-radius-0-tl u-border-radius-0-bl">Last</button>
        </li>
      </ul>
    `}
  </Story>
</Canvas>

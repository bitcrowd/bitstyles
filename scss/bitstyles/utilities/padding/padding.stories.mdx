# Padding

Apply padding of a specified size and direction.

Available combinable variations:

- `0`, `xs`, `s`, `m`, `l`, `xl`, `xxl` sizes
- `top`, `right`, `bottom`, `left`, `x`, `y` directions
- `@m` and `@l` breakpoint suffixes.

# Block

Force an element to `display: block`. This could be to override a default-inline/flow element like a `<span>`:


<Canvas>
  <Story name="Block">
    {`
      <span>Span 1 content</span>
      <span class="u-block">Span 2 content, display: block</span>
      <span>Span 3 content</span>
      <span>Span 4 content</span>
    `}
  </Story>
</Canvas>

Another use is to add `.u-hidden` to “remove” an element from the page, then use a breakpoint-limited version of `.u-block` to reveal it at certain screensizes:

<Canvas isColumn>
  <Story name="Block (small only)">
    {`
      <p>Paragraph 1 content</p>
      <p class="u-hidden u-block@s">Paragraph 2 content, only shown on small viewports</p>
      <p>Paragraph 3 content</p>
    `}
  </Story>
  <Story name="Block (medium +)">
    {`
      <p>Paragraph 1 content</p>
      <p class="u-hidden u-block@m">Paragraph 2 content, only shown on medium-and-up viewports</p>
      <p>Paragraph 3 content</p>
    `}
  </Story>
</Canvas>

# max-width

Specify the `max-width` property of an element. Default configuration provides `max-width: 0` and `100vw` with no breakpoint variations. See [customization](#customization) for details on how to change that.

## Customization

Customize the available max-width by overriding the `$values` Sass map, providing the name to be used for the class, and the value to use.

```scss
@use '~bitstyles/scss/bitstyles/utilities/max-width' with (
  $values: (
    '10rem': 10rem,
  )
);
```

Available breakpoint variables can be specified by overriding the `$breakpoints` Sass list, providing a list of names of breakpoints taken from the global breakpoint list:

```scss
@use '~bitstyles/scss/bitstyles/utilities/max-width' with (
  $breakpoints: (
    'm',
  )
);
```

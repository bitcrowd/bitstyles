# Fixed-ratio

Enforce an aspect ratio on this element and its contents.

The aspect-ratio element takes 100% of its parent elementâ€™s width, then the height is set so the element will maintain the specified ratio. The default ratios inclue only 16:9, defined in the Sass map `$bitstyles-aspect-ratios` under the keys `width` and `height` e.g.

```scss
$bitstyles-aspect-ratios: (
  "16-9": (
    'width': 16,
    'height': 9
  )
)
```

No content may overflow this element.

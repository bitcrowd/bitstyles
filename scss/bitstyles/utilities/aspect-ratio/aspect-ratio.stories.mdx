# Fixed-ratio

Enforce an aspect ratio on this element and its contents.

The aspect-ratio element takes 100% of its parent elementâ€™s width, then the height is set so the element will maintain the specified ratio. The default ratios inclue 1:1 (square), and 16:9 (common for video content). See the [Customization](#customization) section below for details how to change that.

# Customization

Available aspect ratios can be customized by overriding the Sass map `$bitstyles-aspect-ratio-values`, for each providing the name for the class as the key, and a sass map containing `width` and `height`:

```scss
@use '~bitstyles/scss/bitstyles/utilities/aspect-ratio' with (
  $values: (
    '16-9': (
      'width': 16,
      'height': 9,
    ),
    '1-1': (
      'width': 1,
      'height': 1,
    )
  ),
);
```

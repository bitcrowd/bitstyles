# Line-height

Specify `line-height` property on an element. With the default configuration values `1` (smallest) through `6` (largest) are available. These are not available at any breakpoints, but see [customization](#customization) below for details on how change the configuration.

## Customization

Customize the available line-height values by overriding the `$values` Sass map. Provide the name to be used for the class as the key, and the value for the `line-height` property:

```scss
@use '~bitstyles/scss/bitstyles/tools/line-height';
@use '~bitstyles/scss/bitstyles/utilities/line-height' with (
  $values: (
    'large': line-height.get('5'),
    'custom': 1.9827364rem,
  )
);
```

The breakpoints these classes are available at can be customized by overriding the `$breakpoints` variable:

```scss
@use '~bitstyles/scss/bitstyles/utilities/line-height' with (
  $breakpoints: (
    'm',
  )
);
```

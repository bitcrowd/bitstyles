# bg (BackGround color)

Applies a background-color to the element. Default colors are

- `brand-2`
- `black`
- `grayscale-dark-2`
- `grayscale-dark-1`
- `grayscale`
- `grayscale-light-1`
- `grayscale-light-2`
- `grayscale-light-3`
- `white`

See the [customization](#customization) section below for details on how to change available colors.

## Customization

The available background-colors can be customized by overriding the `$values` Sass variable. Use the `var(design-token.get('color', color-name)` function to maintain consistency throughout your project:

```scss
@use '~bitstyles/scss/bitstyles/utilities/bg' with (
  $values: (
    'background': var(design-token.get('color', 'grayscale')),
    'brand-2': var(design-token.get('color', 'brand-2')),
    'grayscale-dark-1': var(design-token.get('color', 'grayscale', 'dark-1')),
    'grayscale': var(design-token.get('color', 'grayscale')),
    'grayscale-light-1': var(design-token.get('color', 'grayscale', 'light-1')),
    'grayscale-light-2': var(design-token.get('color', 'grayscale', 'light-2')),
    'white': var(design-token.get('color', 'grayscale', 'white')),
  )
);
```

The breakpoints these classes are available at can be customized by overriding the `$breakpoints`:

```scss
@use '~bitstyles/scss/bitstyles/utilities/bg' with (
  $breakpoints: (
    'm',
    'l',
  )
);
```

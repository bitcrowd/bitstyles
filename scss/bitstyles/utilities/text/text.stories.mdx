## Font-sizes

Typographic classes that apply font-sizes from the typographic scale. These are useful mainly in two cases:

- when your visual design calls for a specific font-size, but following the correct order of HTML `<h>` elements would result in a different size
- when you need to apply a specific font-size to some other element, such as a button

Note the `.u-h0` has been added as an additional layer above `.u-h1`, use it to acheive extra-large ‘hero’ text.

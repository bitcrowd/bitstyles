# max-height

Specify the `max-height` property of an element. Default configuration provides `100vh` and `stretch` (also known as `fill-available`) as values, with no breakpoint variations. See [customization](#customization) for details on how to change that.

## Customization

Customize the available min-heights by overriding the `$bitstyles-max-height-values` Sass map, providing the name to be used for the class, and the value to use.

```scss
@use '~bitstyles/scss/bitstyles/utilities/max-height' with (
  $values: (
    '10rem': 10rem,
  )
);
```

Available breakpoint variables can be specified by overriding the `$bitstyles-max-height-breakpoints` Sass list, providing a list of names of breakpoints taken from the global breakpoint list:

```scss
@use '~bitstyles/scss/bitstyles/utilities/max-height' with (
  $breakpoints: (
    'm',
  )
);
```

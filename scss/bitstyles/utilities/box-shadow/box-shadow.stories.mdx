## Customization

The available drop-shadows and their names can be customized by overriding the `$bitstyles-drop-shadow-values` Sass variable. Use the `shadows.get()` helper from `tools/shadows` to access the shadows youâ€™ve setup in your global design tokens (see `settings/shadows`), providing `filter` as the format (drop-shadows are applied using CSS filters):

```scss
$bitstyles-display-values: (
  'default': shadows.get('default', 'filter'),
);
```

The breakpoints these classes are available at can be customized by overriding the `$bitstyles-drop-shadows-breakpoints` variable:

```scss
$bitstyles-drop-shadows-breakpoints: ('s', 'm', 'xl');
```

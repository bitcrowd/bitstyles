# Box-shadow

Apply a CSS box-shadow to the element. By default this is selected from the standard set of shadows, but you can specify any shadow (see the [“Customization”](#customization) section below).

## Customization

The available box-shadows and their names can be customized by overriding the `$values` Sass variable. Use the `shadows.get()` helper from `tools/shadows` to access the shadows you’ve setup in your global design tokens (see `design-tokens/shadows`), providing `css` as the format:

```scss
@use '~bitstyles/scss/bitstyles/tools/shadows';
@use '~bitstyles/scss/bitstyles/utilities/box-shadow' with (
  $values: (
    'brand-1': shadows.get('brand-1', 'box'),
  )
);
```

You can also specify shadows directly, using the correct format for the CSS `box-shadow` property e.g.

```scss
@use '~bitstyles/scss/bitstyles/utilities/box-shadow' with (
  $values: (
    'brand-1': 0 0 1rem 0.5rem rgba(0, 0, 0, 0.2),
    0 0 2rem 1rem rgba(0, 0, 0, 0.2),
  )
);
```

The breakpoints these classes are available at can be customized by overriding the `$breakpoints` variable, providing a list of breakpoint names from your globally-defined breakpoints:

```scss
@use '~bitstyles/scss/bitstyles/utilities/box-shadow' with (
  $breakpoints: (
    's',
    'm',
    'l3',
  )
);
```

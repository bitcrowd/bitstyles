# min-height

Specify the `min-height` property of an element. Default configuration provides `min-height: 0` and `min-height: stretch` (also known as `fill-available`), with no breakpoint variations. See [customization](#customization) for details on how to change that.

## Customization

Customize the available min-heights by overriding the `$values` Sass map, providing the name to be used for the class, and the value to use.

```scss
@use '~bitstyles/scss/bitstyles/utilities/min-height' with (
  $values: (
    '10rem': 10rem,
  )
);
```

Available breakpoint variables can be specified by overriding the `$breakpoints` Sass list, providing a list of names of breakpoints taken from the global breakpoint list:

```scss
@use '~bitstyles/scss/bitstyles/utilities/min-height' with (
  $breakpoints: (
    'm',
  )
);
```

# Gap

Specify the `gap` property of an element. This applies to flex and grid layouts, though right now the flex version is not as well supported. Use autoprefixer to ensure the older `grid-gap` version of the property is used in your final CSS.

By default, available sizes are `xs`, `s`, `m`, and `l`, and the classes are available at the `m` and `xl` breakpoints. See the [customization](#customization) section below for details on how to change those settings.

## Customization

The available sizes can be customized by overriding `$sizes`. Provide the name to be used for the class as the key, and the size of gap. Note that if you want to keep the existing values, youâ€™ll need to copy them.

```scss
@use '~bitstyles/scss/bitstyles/utilities/gap' with (
  $sizes: (
    's4': size.get('s4'),
    's2': size.get('s2'),
    'l2': size.get('l2'),
    'l3': size.get('l3'),
  )
);
```

The breakpoints these classes are available at can be customized by overriding `$breakpoints` Sass list:

```scss
@use '~bitstyles/scss/bitstyles/utilities/gap' with (
  $breakpoints: (
    'm',
    'xl',
  )
);
```

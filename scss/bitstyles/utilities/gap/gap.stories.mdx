# Gap

The gap classes specify gaps on your grid elements â€” space between each grid item, but no space around them (you need to handle the space outside yourself).

By default, `s`, `m`, and `l` sizes are available. This can be customized by overriding `$bitstyles-gap-sizes` e.g.

```scss
$bitstyles-gap-sizes: (
  'my-gap-size': 5rem
);
```

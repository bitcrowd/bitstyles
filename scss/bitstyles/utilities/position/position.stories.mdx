## Customization

The available positioning classes can be configured by overriding `$bitstyles-position-values` Sass map, providing the classname as the key, and the size as value:

```scss
@use '~bitstyles/scss/bitstyles/utilities/position' with (
  $values: (
    'relative': relative,
    'absolute': absolute,
    'fixed': fixed,
    'sticky': sticky,
    'static': static,
  )
);
```

To customise the breakpoints these classes are all available at, override `$bitstyles-position-breakpoints` with a list of the breakpoint names from the global settings.

```scss
@use '~bitstyles/scss/bitstyles/utilities/position' with (
  $breakpoints: (
    'm',
    'l',
  )
);
```

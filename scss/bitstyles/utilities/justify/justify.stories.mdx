# Justify classes

## Justify

Specify `justify-content` property on an element.

Available with `between`, `start`, and `end` values by default, and at the `m` breakpoint. See [customization](#customization) below for details on how change the settings.

Flex and Grid share many alignment properties. These will have no effect on elements to which you havenâ€™t given one of those classes (`.u-flex`, `.u-grid`).

The `u-justify-` classes set set `justify-content` to align flex children to the respective different edges on the cross-axis. This only affects layout when the combined elements are smaller than the available cross-axis:

## Responsiveness

These classes are available at the `m` breakpoint with the default configuration.

## Customization

Customize the available justify-content values by overriding the `$bitstyles-justify-content-values` Sass map. Similarly for `justify-items` and `justify-self`: `$bitstyles-justify-items-values` `$bitstyles-justify-self-values`.

Provide the name to be used for the class as the key, and the value for the property:

```scss
@use '~bitstyles/scss/bitstyles/utilities/justify' with (
  $content-values: (
    'between': space-between,
    'end': end,
    'start': start,
  ),
  $items-values: (
    'stretch': stretch,
    'end': end,
    'start': start,
  )
);
```

The breakpoints these classes are available at can be customized by overriding the `breakpoints` variables `$bitstyles-justify-content-breakpoints`, `$bitstyles-justify-items-breakpoints`, and `$bitstyles-justify-self-breakpoints`

```scss
@use '~bitstyles/scss/bitstyles/utilities/justify' with (
  $content-breakpoints: (
    'm',
  ),
  $self-breakpoints: (
    'm',
  )
);
```

# Fixed-ratio

Enforce an aspect ratio on this element and its contents.

The fixed-ratio element takes 100% of its parent elementâ€™s width, then the height is set so the element will maintain the specified ratio. The default ratio is 16:9, defined in the Sass map `$fixed-ratio-default-ratio` under the keys `width` and `height` e.g.

```scss
$fixed-ratio-default-ratio: (
  'width': 16,
  'height': 9
)
```

No content may overflow this container.

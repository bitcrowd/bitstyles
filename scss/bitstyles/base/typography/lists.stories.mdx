# Typography

## Definition list

Used to define terms, in term/definition pairs similar to a dictionary. A `<dl>` is a great choice when you have e.g. some information about a user or other entity to display.

<Canvas>
  <Story name="dl">
    {`
      <dl>
        <div>
          <dt>Name</dt>
          <dd>Jelly Marshmallow</dd>
        </div>
        <div>
          <dt>Email</dt>
          <dd>jelly@marshmallow.com</dd>
        </div>
      </dl>
    `}
  </Story>
</Canvas>

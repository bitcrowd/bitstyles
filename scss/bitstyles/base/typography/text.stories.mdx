# Typography

## Common text elements

### Paragraphs

<Canvas>
  <Story name="p">
    {`
      <p>Tiramisu pie marshmallow jujubes brownie jelly beans cheesecake. Oat cake cake topping chocolate pie jujubes cupcake. Tart liquorice sugar plum sweet.</p>
    `}
  </Story>
</Canvas>

### Address

Address element. Often used to indicate authorship and contact information.

<Canvas>
  <Story name="Address">
    {`
      <address>
        Name<br/>
        Address line 1<br/>
        Address line 2<br/>
        City<br/>
        Country<br/>
        Postal code
      </address>
    `}
  </Story>
</Canvas>

### Blockquote

Quoted text, ideally with citation if the quote is not taken from the current document, or from a person being e.g. interviewed.

<Canvas>
  <Story name="Blockquote">
    {`
      <blockquote>
        <p>Tiramisu pie marshmallow jujubes brownie jelly beans cheesecake. Oat cake cake topping chocolate pie jujubes cupcake. Tart liquorice sugar plum sweet.</p>
        <cite><a href="http://www.cupcakeipsum.com">cupcakeipsum.com</a></cite>
      </blockquote>
    `}
  </Story>
</Canvas>

## Some other text elements

These are elements that are commonly used, but rarely all on a single project. They are included here to ensure they are tested. For an exhaustive list of HTML elements, look to the [MDN elements reference](https://developer.mozilla.org/en-US/docs/Web/HTML/Element).

### strong

[Urgent or important](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/string).

<Canvas>
  <Story name="Strong">
    {`
      <strong>Jelly cake marshmallow</strong>
    `}
  </Story>
</Canvas>

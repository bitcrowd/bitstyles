# Tabs

The full tab UI uses a list of `.o-button--tabs` for the tabs, with minimal styling of the tabpanels themselves â€” they are just containers for the content you need to render inside.

Take note of all the aria attributes that tie the tabs to their tabpanels, and vice versa. These will be needed for keyboard & screenreader users, and for the JS that will control the tab UI (not yet provided).

Also note the differences in attributes of selected/deselected tabs (`aria-selected` is switched) and selected/deselected tabpanels (`hidden` attribute is added/removed).

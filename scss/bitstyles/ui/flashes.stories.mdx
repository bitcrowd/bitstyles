# Flashes

Inform the user of a global or important event, such as may happen after a page reload. There are several variants, which use color, iconography, and html attributes to indicate severity.

See [Atoms/Flash](/docs/atoms-flash-overview--page) for more details on how to use Flashes.

## Flashes in forms

If a flash is used to give feedback to a form submission, place the flash close to the form in the HTML, and visually. See [UI/Forms](/docs/ui-data-forms--login-form-with-errors) and [Atoms/Cards](/docs/atoms-card--card-header) for more examples of Flashes in use.

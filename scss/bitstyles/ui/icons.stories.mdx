# Icons

The included SVG icons can be used in two ways:

- \[Recommended\] Host it as a normal image `.svg` file, then reference specific icons inside with SVG `<use>` elements
- Inlined into the top of the HTML and hidden, then referenced with SVG `<use>` elements

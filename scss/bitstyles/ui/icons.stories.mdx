# Icons

Icons must always be `aria-hidden` and have some text content as a near-sibling that expresses the same as the icon. That text can be given the `u-sr-only` class to hide it from sight, if needed.

The included SVG icons can be used in two ways:

- \[Recommended\] Host it as a normal image `.svg` file, then reference specific icons inside with SVG `<use>` elements
- Inlined into the top of the HTML and hidden, then referenced with SVG `<use>` elements

## Icon sizes

By default, icons inherit their size from the surrounding text i.e. `width: 1em; height: 1em`. Use the modifier classes `a-icon--<size>` to override this:

- `.a-icon--s`
- `.a-icon--m`
- `.a-icon--l`
- `.a-icon--xl`
